# Bit a Bit - IP Calculator

A responsive IP calculator and subnetting tool developed as an academic project for my Networking studies. This application allows users to perform network calculations, understand subnetting, and visualize address ranges.

## 🚀 Features

- **Network Calculation:** Quickly calculate network addresses, broadcast addresses, and valid host ranges.
- **Subnetting Support:** Easily handle different CIDR prefixes.
- **Responsive Design:** Fully optimized for mobile and desktop screens using Tailwind CSS.
- **Dark/Light Mode:** Toggleable theme to improve user experience.

## 🛠️ Technologies Used

- **Frontend:** React, Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Language:** TypeScript

## 📦 How to run locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/bit-a-bit-ip-calculator.git
   ```

2. **Navigate to the project folder:**


```bash
cd bit-a-bit-ip-calculator
```

3. **Install dependencies:**

```bash
npm install
```

4. **Start the development server:**

```bash
npm run dev
```
